# Ride App Backend

Backend API cho thành viên 1: User & Auth.

## Cài đặt
```bash
npm install
```

## Chạy
```bash
npm run dev
```

## Environment
Copy `.env.example` thành `.env` và cấu hình thông tin SQL Server.

## API
- `POST /api/auth/register`
- `POST /api/auth/login`
- `GET /api/auth/me`
- `GET /api/health`
